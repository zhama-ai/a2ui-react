# Screenshots

This directory contains screenshots for the README documentation.

## How to Generate Screenshots

1. Run the basic-demo example:
   ```bash
   cd examples/basic-demo
   npm install
   npm run dev
   ```

2. Open http://localhost:5174 in your browser

3. Take screenshots of the following scenes:

### welcome.jpg
- Type `hello` in the chat
- Wait for the AI to generate the welcome screen with topic cards
- Take a screenshot of the entire application window

### learning.jpg
- Click on any "开始学习" button
- Wait for the learning path to be generated
- Take a screenshot showing the structured content

### answer.jpg
- Click "直接开始练习" button
- Wait for the quiz questions to appear
- Take a screenshot showing the interactive questions

## Screenshot Requirements

- Format: JPG or PNG
- Dimensions: Approximately 1200x800 pixels (or higher for retina displays)
- Quality: High resolution for documentation
- Content: Show the complete UI with both chat panel and workspace panel
- Aspect Ratio: Maintain 3:2 or 16:10 for consistent display
