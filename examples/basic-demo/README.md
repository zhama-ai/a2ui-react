## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start development server
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173) to view the demo.

### Build

```bash
# Build for production
pnpm build

# Preview production build
pnpm preview
```

## How It Works

1. **User Input**: Type a message in the chat panel
2. **Mock Agent**: The mock agent simulates an AI response with A2UI data
3. **UI Rendering**: The A2UIRoot component renders the dynamic UI
4. **Interaction**: Users can interact with the generated UI components

## A2UI Protocol

This demo uses the A2UI protocol to generate dynamic UIs. The protocol defines a JSON structure that maps to React components:

```json
{
  "type": "card",
  "props": {
    "title": "Example Card",
    "description": "This is generated by A2UI"
  },
  "children": [...]
}
```

## Try These Messages

- "Hello" - Get a welcome card with buttons
- "Show me items" - See a list of interactive cards
- "Form" - Get a form with input fields
